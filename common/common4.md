4. Алгоритм поиска максимума в списке, ссылается на алгоритм сравнение двух чисел в списке.

**Алгоритм 1: Найти максимум в списке**
Ввести список A
Пусть max = A[0]
Для каждого элемента x из списка:
Вызвать алгоритм "Сравнение двух чисел" для max и x
Если x больше, обновить max
Вывести max

**Алгоритм 2: Сравнение двух чисел**
Ввести a и b
Если a > b вернуть a
Иначе вернуть b
