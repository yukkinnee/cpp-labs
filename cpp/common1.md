# Лабораторная работа 1

  Процессор - главный блок всего компьютера, его мозг, который выполняет все основные операции: обработка данных, выполнение инструкций программ
Память - это место для хранения данных. Бывают разных типов: Оперативная(быстрая, но временная), Постоянная(SDD HDD - медленнее, но сохраняет данные) Кэш-память(встроена в процессор) Регистры(ячейки памяти внутри самого процессора)

Отличия:
RAM - большая по объему, используется для хранения данных программ во время работы
Регистры - ячейки памяти внутри самого процессоры, в несколько раз быстрее. Хранят промежуточные данные


Инструкция - это команда процессору что ему делать. К примеру: сложить числа, переместить данные и тд
Instruction Set Architecture (ISA) - это язык на котором процессор понимает команды
Почему инструкциям необходим идентификатор - чтоб процессор понимал какую именно операцию ему следует выполнить в данный момент времени
Машинный код - это инструкция в двоичном коде, который понимает процессор
Адрес памяти - уникальный номер ячейки памяти, где хранится какая-то определенная информация
Байт - минимальная стандартная единица хранения данных, равная 8 битам
